# Crank & Torque Auto (demo)

A demo website for a fictional Louisville auto repair shop. Dark industrial design: charcoal base, safety-orange accent, condensed display type, hazard-stripe details, and tap-to-call throughout.

Live at [davin-willis.github.io/auto-repair-demo](https://davin-willis.github.io/auto-repair-demo/). Part of my client web work at [davinwillis.dev](https://davinwillis.dev).

## AI-assisted development

Built with AI-assisted development. All architecture, design, and implementation decisions are my own.

## Notes

The business is fictional. All names, phone numbers, addresses, and details are placeholders, and photo slots are clearly marked placeholders.

## Stack

- Next.js (App Router, TypeScript, static export)
- Tailwind CSS v4
- Framer Motion (scroll reveals, respects prefers-reduced-motion)
- Fonts: Anton, Barlow, IBM Plex Mono via next/font
- Deployed to GitHub Pages via GitHub Actions

## Run locally

```bash
npm install
npm run dev
```
